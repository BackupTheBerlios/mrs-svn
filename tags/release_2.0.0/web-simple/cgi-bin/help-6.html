The results

If you selected 'All Databanks' in the popup the result of a search will be a list of databanks containing hits for this query. The list is built up in real time and once it is finished a summary of the statistics will be given.

If the search you did yielded less than 1000 hits per databank the number shown in this list is exact. However, if the number is larger than 1000 it might be an estimate. If a number is an estimate it will be preceded by a ≈ sign. This holds for all other places where number of hits are displayed.

In the list with databanks you can click one of them and this will repeat the search in that particular databank only, just like you would have selected that databank from the popup menu when doing the search.

The results per databank list

The page you will see when doing a search in one databank only has several features that can help you interpreting the data. First of all there is a list containing two columns, one with the ID for each hit and the second contains a short description.

Move the mouse over the screenshot below to get an explanation of the items available, the explanation should appear in this box.

Here you see which hits are displayed and the total number of hits available. Again, if the total number is higher than 1000 the total might be an estimate in which case the number is preceded by a ≈ sign.

If there are more hits than fit in the current setting for the maximum number of hits per page the buttons Next and Prev allow you to browse through the hits page by page.

This popup button allows you to display the hits as summary (which is the default), in FastA format (if available) and as complete entries.

The number of lines in the list is 15 by default but can be changed by typing a diffent number in this edit box and then pressing the Enter key on the keyboard.

This pair of buttons allow you to export the data in a form without all the HTML markup. Using the popup button you can choose between displaying the data in a browswer window or sending the output to a file.
